AuthComponent

o componente AuthComponent utiliza do componente LayoutAuthComponent para renderizar o leu layout, mas nada deve ser alterado dentro do LayoutAuthComponent, tudo é setado dentro do póprio AuthComponent.

Margin

O componente LayoutAuthComponent por padrão tem um @input chamdo margin que é uma string literal *"full"* ou um numero;

FullNúmero

O componente tocará as bordas do navegador.

<app-layout-auth [margin]="full"></app-layout-auth>

O componente tocará as bordas do navegador.

<app-layout-auth [margin]="20"></app-layout-auth>

Banner Lateral

componente LayoutAuthComponent pode renderizar com ou sem um banner lateral

SEM BANNERCOM BANNER

apenas chamando o componente LayoutAuthComponent

<app-layout-auth></app-layout-auth>

Adicine uma tag div com atributo banner


<app-layout-auth>
    <div banner></div>
</app-layout-auth>

Posição Banner Lateral

O componente LayoutAuthComponent por padrão tem um @input chamdo ladoBanner que é uma string literal *'direito'|'esquerdo'*

DIREITOESQUERDO

O banner renderizará do lado direito


<app-layout-auth ladoBanner="direito">
    <div banner></div>
</app-layout-auth>

O banner renderizará do lado esquerdo


<app-layout-auth ladoBanner="esquerdo">
    <div banner></div>
</app-layout-auth>

Banner Lateral Estilização

o div com atributo banner pertence ao componente AuthComponent e por isso sua estilização deve ser feita nele, seja por classes, pela tag style ou pelo file auth.component.scss

Clique aqui para ver todas as clases de estilização

EXEMPOS
 

<app-layout-auth>
    <div banner class="color-light"></div>
</app-layout-auth>
                        

<app-layout-auth>
    <div banner style="color: #fffff"></div>
</app-layout-auth>
                        
                

auth.component.scss
[banner]: { color: #fffff; }

Banner Rotativo

O banner lateral tem a opção de ser um banner rotativo (carrossel) com infinitos banners. O numero de divs com atributo banner virará automaticamente um banner rotativo

EXEMPLO
 

<app-layout-auth>
    <div banner></div>
    <div banner></div>
    <div banner></div>
 </app-layout-auth>
